VERBOSITY

If you provide swift-t -V, swift-t uses stc -V .
If you provide swift-t -V -V, swift-t also uses turbine -V .

ENVIRONMENT VARIABLES

  STC_LOG_TRACE
    If stc logging is enabled, this enables trace-level logging
  STC_JVM_FLAGS
     Additional flags to pass to JVM for compilation
